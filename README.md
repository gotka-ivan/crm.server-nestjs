# crm.server-nestjs

Сервер на Nest.js и Api на основе Swagger

## Инициализация проекта

```
yarn install
```

### Запуск сервера с горячей-перезагрузкой

```
yarn start:dev
```

### Запуск для развертывания

```
yarn start
```

### Проверка и исправление файлов

```
yarn lint
```

#### Рекомендации

- Работать исключительно со сборщиком Yarn
